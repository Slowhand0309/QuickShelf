## Features

- 📁 Instant access to any folder's contents
- 📦 Drag & drop support
- ⚡ Lightweight and fast
- ⌨️ Hotkey Support
- 🔄 Start at login

## Installation
Download the latest release from the [here](https://quickshelf-app.slowlab.dev/)

## Requirements
- macOS 14.6 or later
- Apple Silicon or Intel processor
- 64-bit architecture
